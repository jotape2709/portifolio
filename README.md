## Product Experience

The interface is intentionally structured as a command center:

- **Boot-like landing state** with system status language.
- **Command navigation** to open modules like profile, capabilities, trajectory, projects, and contact.
- **Subtle responsive behavior** (highlight pulses and contextual state changes) to make the UI feel alive without noise.
- **Bilingual support** (Português BR / English) with polished copy in both languages.
- **Dual visual modes** (dark / light), each designed as a first-class experience.

---

## Technical Positioning

Built from scratch with a practical and security-minded approach:

- No unnecessary external runtime dependencies.
- Minimal, auditable surface area (HTML/CSS/JS only).
- Safe external linking (`rel="noopener noreferrer"`).
- No unsafe dynamic HTML insertion.
- Clear separation of structure, presentation, and behavior.
