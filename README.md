# JPML // C2 Terminal

Portfólio interativo no estilo terminal C2 blood-red — construído do zero, sem frameworks, sem bullshit.

---

## O que é isso

É o meu portfólio. Mas feito como um terminal de comando, porque um site genérico de "olá, sou fulano" não me representa.

Você abre, vê uma tela de boot, e cai num terminal funcional. Você digita comandos. O ambiente responde. É interativo de verdade.

---

## Como usar

```
help            → lista os comandos disponíveis
open profile    → quem sou eu
open skills     → o que eu sei fazer
open experience → de onde vim
open projects   → o que já construí
open contact    → como me encontrar
whoami          → resumo rápido
ls              → lista os módulos disponíveis
clear           → limpa o terminal
```

Tem uns easter eggs também. Descubra.

---

## Stack

HTML + CSS + JavaScript puro. Sem dependências. Sem build. Abre no browser e funciona.

- Tema escuro blood-red com efeito CRT
- Tema claro disponível (botão ☾/☼)
- Bilíngue (PT-BR / EN)
- Boot sequence com ASCII art
- Terminal interativo com múltiplos comandos
